Internet Explorer (anteriormente Microsoft Internet Explorer y Windows Internet Explorer, usualmente abreviado a IE) es una serie descontinuada de navegadores web gráficos desarrollados por Microsoft para Microsoft Windows desde 1995. Se lanzó por primera vez como parte del paquete complementario Plus! para Windows 95 ese año.
