Notas y referenciasEditar

  1. ↑ Belfiore, Joe (6 de diciembre de 2018). «Microsoft Edge: Making the web better through more open source collaboration» (html). Microsoft Windows Blog (en inglés). Archivado desde el original el 7 de diciembre de 2018. Consultado el 7 de diciembre de 2018. «We will move to a Chromium-compatible web platform for Microsoft Edge on the desktop. Our intent is to align the Microsoft Edge web platform simultaneously (a) with web standards and (b) with other Chromium-based browsers. This will deliver improved compatibility for everyone and create a simpler test-matrix for web developers. » 
  2. ↑ Internet Explorer es el único navegador que soporta el Modelo de Objeto Componente (también conocido como ActiveX) de forma nativa. El embebido de ActiveX en Internet Explorer añade funcionalidad al navegador, aunque también puede conducir a infecciones de virus, troyanos y spyware.
  3. ↑ Incluida en la versión 10, para las anteriores la corrección ortográfica puede ser añadida instalando ieSpell.
  4. ↑ Internet Explorer la incluye a partir de la versión 7.
  5. ↑ El zoom de página es distinto al cambio de tamaño del texto, puesto que cambia el tamaño no sólo de los caracteres, sino también de los objetos multimedia y la vista.
  6. ↑ a b c Mozilla, Mozilla Firefox, Netscape y SeaMonkey soportan bloqueo de imágenes sencillo basado en el nombre de dominio. Se puede añadir un bloqueo de anuncios más avanzado basado en expresiones regulares en Mozilla y Mozilla Firefox con software externo como la extensión AdBlock.
  7. ↑ a b c Bug 4821 – Full zooming not functional (images, objects as well as text) (page zoom like opera)
  8. ↑ El soporte de eventos de ratón puede ser añadido instalando Easy Go Back.
  9. ↑ a b c d El soporte de eventos de ratón puede ser añadido instalando extensiones como All-in-One Gestures (sólo Firefox) y Mouse Gestures.
  10. ↑ Doug Turner, el principal desarrollador de Minimo, ha introducido navegación espacial en algunas versiones especiales de Firefox. Se puede hacer que sea una característica por defecto de Firefox.
  11. ↑ Firefox trabaja con varios lectores de pantalla como JAWS y Microsoft SAPI 5 TTS a través de extensiones.
  12. ↑ CSS2 Revisión 1, una recomendación candidata del W3C, es la versión actual de CSS. CSS2 no está apenas soportando, por lo que no se considera aquí. CSS3 sólo es un proyecto en estos momentos. Para información más detallada, por favor vea comparativa de motores de navegación (CSS).
  13. ↑ XHTML está basado en HTML pero es una aplicación de XML, lo que significa que XHTML debe ser más estricto que su código equivalente escrito en HTML. XHTML debería poderse leer desde cualquier parser (analizador sintáctico) XML, pero por razones de retrocompatibilidad también puede ser analizado sintácticamente como HTML; esta tabla sólo cita los navegadores que son capaces de analizar XHTML como si fuera XML. Para una información más detallada, por favor vea comparativa de motores de navegación (XHTML).
  14. ↑ Internet Explorer 6 soporta varios selectores y propiedades de CSS2, pero se sabe que tiene un número de errores significantemente mayor que otros navegadores. El más notable de éstos es el error de modelo de cajas, que afecta a todas las versiones de Internet Explorer para Windows antes de IE6, y aún es el modelo de cajas predeterminado en IE6.
  15. ↑ La prueba del soporte de los estándares web no llega a más de 20%, se puede comprobar con Internet Explorer en el test de Acid3.
  16. ↑ a b KDE usa Akregator (componente de Kontact) para leer extensiones RSS y Atom, o también puede usar KNewsTicker.
  17. ↑ a b c XForms es soportado de forma experimental en las versiones construidas después del 28 de enero de 2005. Requiere la instalación de una extensión.
  18. ↑ a b Muchos navegadores han evitado a propósito el soporte para e-mail y grupos de noticias, ya que lo reservan para sus propios programas clientes de e-mail. Para una comparativa de esos programas, vea comparativa de clientes de correo electrónico.
  19. ↑ a b c Los dominios IDN se muestran como punycode por defecto por razones de seguridad.
  20. ↑ a b El soporte para IRC puede ser añadido instalando ChatZilla.
  21. ↑ Gopher es soportado a través de servidores proxy.
  22. ↑ SVG aquí se refiere a SVG 1.1 "Completo". Estos son dos perfiles conocidos como SVG 1.1 Tiny (Minúsculo) y SVG 1.1 Basic (Básico), que están pensados para agentes de usuario con capacidades limitadas.
  23. ↑ La mayoría de navegadores soportan PDF instalando un plugin de Adobe que visualiza el archivo en una ventana del navegador. Aquí también hay listados algunos navegadores que también soportan PDFs en línea dentro de otros documentos de hipertexto (como dentro del elemento <<timg de HTML). notar que PDF (estrictamente hablando) no es un formato de imagens, sino un documento de texto rico que puede poseer scripts, que contiene diferentes tipos de contenido multimedia, incluyendo gráficos vectoriales e imágenes de mapa de bits, audio, vídeo, formularios, enlaces de hipertexto a intra e inter-documentales y un listado de contenido jerárquico. el formato también es el formato de visualización nativa bajo Mac OS X.
  24. ↑ Internet Explorer no soporta el mostrado progresivo de los JPEG progresivos.
